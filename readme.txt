# Rock, Paper, Scissors Game

## Overview

This is a simple Rock, Paper, Scissors game implemented using HTML, CSS, and Vanilla JavaScript. It allows users to play the classic Rock, Paper, Scissors game against the computer.

## How to Play

1. Open the `index.html` file in your web browser.
2. You will see the game interface with buttons for Rock, Paper, and Scissors.
3. Click on one of the buttons to make your choice.
4. The computer will randomly choose its move.
5. The result of the game will be displayed on the screen.

## Files and Structure

- `game.html`: The main HTML file containing the game structure and buttons.
- `game.css`: The CSS file for styling the game interface.
- `game.js`: The JavaScript file containing the game logic.

## Customization

Feel free to customize the game by modifying the styles in `game.css` or adjusting the game logic in `game.js`. You can add new features, improve the UI, or enhance the gameplay.
